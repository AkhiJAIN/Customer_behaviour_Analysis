# Customer Behaviour Analysis Dashboard

This project is an end-to-end Data Analytics project where customer shopping data is cleaned, stored in SQL Server, analyzed using SQL queries, and visualized using Power BI.

## 📌 Project Workflow

1. Data Collection
2. Data Cleaning using Python & Pandas
3. Feature Engineering using Pandas
4. Store data in MS SQL Server
5. Data analysis using SQL queries
6. Dashboard creation using Power BI

---

## 🛠 Tools & Technologies Used

- Python
- Pandas
- MS SQL Server
- SQL
- Power BI
- DAX

---

## 📂 Project Steps

### 1. Data Cleaning (Python + Pandas)

- Removed null values
- Fixed data types
- Created new columns (Age Group, etc.)
- Prepared data for database

### 2. Feature Engineering

- Created Age Group column
- Created category summaries
- Calculated purchase statistics

### 3. Database Creation (MS SQL Server)

- Created database in SQL Server
- Created table
- Inserted cleaned data from Python

### 4. SQL Analysis

Used SQL queries to analyze:

- Total customers
- Purchase amount
- Category wise sales
- Gender wise analysis
- Season wise sales
- Payment method usage

### 5. Power BI Dashboard

Created interactive dashboard with:

- KPI Cards
- Category Sales Chart
- Age Group Revenue Chart
- Season Wise Sales
- Payment Method Table
- Gender Analysis
- Subscription Analysis
- Filters (Category, Gender, Shipping Type)
